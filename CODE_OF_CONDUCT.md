# Code of Conduct

Project Serendib is a **national digital public good** initiative for Sri Lanka’s education and enterprise ecosystem. We welcome contributions from students, researchers, educators, and the public.

## Our standards

We expect:

- Be respectful, inclusive, and constructive.
- Prioritize **safety**: do not contribute harmful content, harassment, hate, or incitement.
- Protect privacy: **never** submit personally identifiable information (PII), credentials, or private documents.
- Respect provenance and licensing: document sources and permissions for all external content.
- Prefer quality over quantity: submit small, reviewable improvements.

## Data-specific rules

- Do not add copyrighted content without explicit permission.
- For sensitive domains (education, law, health), prefer grounded sources and clear uncertainty.
- Tag synthetic content clearly as `synthetic` with provenance; synthetic data requires human verification before release.

## Enforcement

Maintainers may remove content or restrict participation when these rules are violated.

## Reporting

If you spot harmful content, PII, or licensing problems, open a GitHub issue using the appropriate template (or contact maintainers privately if needed).
