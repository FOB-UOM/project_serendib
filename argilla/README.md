# Argilla (student verification)

This folder contains a **Hugging Face Space-ready** scaffold to run student verification tasks (quality, safety, language, domain tagging).

## What students do

- Review proposed instruction dialogues
- Mark issues (factuality, harmful content, PII, formatting)
- Approve or request changes

## Space scaffold

See `argilla/space/` for a minimal Gradio app that connects to Argilla and presents verification queues.
