# OCR Pipeline (`ocr-pipeline/`)

Starter OCR utilities for Sinhala-centric document ingestion.

Planned components:

- **Tesseract Sinhala** OCR for scanned PDFs/images
- **TrOCR**-based OCR for higher accuracy (GPU optional)
- Post-processing: normalization, dehyphenation, deduplication, metadata capture

Keep this folder runnable end-to-end for newcomers (one-command examples).
