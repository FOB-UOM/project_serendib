# Evaluation

This folder holds the evaluation scaffolding for `Lanka-Instruct-v1`.

## Goals

- Track improvements in **Sri Lankan cultural relevance** (per the proposal).
- Keep evaluation reproducible and lightweight.

## Planned contents

- `prompts/`: evaluation prompts and rubrics
- `sets/`: curated evaluation datasets (small, human-reviewed)
- `run_eval.py`: harness to run baseline vs fine-tuned model comparisons
